# 单词闯关游戏 - 产品需求文档 (PRD)

## 📋 产品概述

### 产品名称
单词闯关 - 在线英语学习游戏

### 产品定位
一款面向3年级到高三学生的互动式英语单词学习游戏，通过游戏化的方式提升英语词汇学习效果，专注于在线双人对战模式，提供更具竞争性和互动性的学习体验。设备支持PC、iPad、手机等多平台。

### 目标用户
- **主要用户**: 3年级到高三学生
- **次要用户**: 英语学习爱好者、教师、家长

### 核心价值
- 🎮 **游戏化学习**: 将枯燥的单词记忆转化为有趣的对战体验
- 🏆 **竞技互动**: 通过真实双人对战激发学习动力和竞争意识
- 📊 **个性化学习**: 根据年级提供适配的词库内容
- 📱 **随时随地**: 支持多设备跨平台对战，随时随地学习
- 🤖 **智能匹配**: AI对手备选机制，确保随时可以开始游戏

## 🎯 功能架构

### 1. 游戏模式

#### 1.1 在线对战模式
**功能描述**: 两名玩家实时对战，通过小游戏形式竞技

**核心流程**:
```
开始对战 → 信息输入 → 匹配等待 → 对战进行 → 结果展示 + 数据统计
```

**对战小游戏**:
- **警察抓小偷** (第一关): 追逐类游戏，中译英填空，答对前进，答错不动，警察追上小偷或小偷到达终点决定胜负
- **登天梯竞赛** (第二关): 攀爬类游戏，英译中选择，答对上升2格，答错下降3格，先到达20层获胜
- **拔河竞赛** (第三关): 拔河类游戏，听写模式，答对拉绳2格，答错对方得1格，将绳子拉过己方7格线获胜

### 2. 核心功能模块

#### 2.1 用户系统
- **信息收集**: 昵称、年级、头像选择
- **数据存储**: 本地存储用户偏好和游戏记录
- **年级适配**: 根据年级自动匹配对应词库

#### 2.2 题库系统
- **词库覆盖**: 3年级到高中完整词库
- **智能抽取**: 随机选择题目，避免重复
- **动态管理**: 实时补充题库，确保题目供应
- **质量保证**: 所有词条都有准确的中文释义

#### 2.3 音频系统
- **TTS语音**: 使用Edge TTS提供高质量英文发音
- **音效反馈**: 丰富的游戏音效增强体验
- **音量控制**: 支持音效开关和音量调节

#### 2.4 匹配系统
- **智能匹配**: 基于年级的自动匹配，确保公平竞争
- **实时通信**: Firebase实现低延迟对战通信
- **AI对手备选**: 30秒无匹配时自动提供AI助手
- **跨设备支持**: 支持不同设备间的实时对战

## 🎮 游戏机制设计

### 1. 双人模式机制

#### 1.1 匹配规则
- **年级匹配**: 优先匹配相同年级玩家
- **等待时间**: 最长等待60秒
- **AI对手备选**: 无真实对手时自动提供AI助手
- **跨设备匹配**: 支持不同设备间的实时匹配

#### 1.2 胜负规则
- **三局两胜**: 先赢得2关的玩家获胜
- **时间限制**: 单局最长15分钟
- **超时判定**: 超时按当前小游戏状态判定胜负
- **无时间限制**: 移除15秒答题倒计时，专注竞速对抗

### 2. 小游戏机制

#### 2.1 警察抓小偷
- **游戏规则**: 警察追捕小偷，双方在跑道上移动
- **移动规则**: 答对前进1步，答错后退1步
- **胜利条件**: 警察追上小偷或小偷到达终点

#### 2.2 登天梯竞赛
- **游戏规则**: 双方攀爬20层天梯
- **移动规则**: 答对上升2格，答错下降3格，最低0格
- **胜利条件**: 先到达20层或时间结束时层数更高

#### 2.3 拔河竞赛
- **游戏规则**: 拔河比赛，绳子中心点左右移动
- **移动规则**: 答对拉绳2格，答错对方得1格
- **胜利条件**: 将绳子拉过己方8格线
